Require: node >=v14.17.1

```bash
$ git clone https://github.com/infra-geo-ouverte/igo2-api.git
$ cd igo2-api

# Install dependencies
$ npm install

# Check files and launch dev instance
$ npm start
# Open your browser at http://localhost:5000/

# Build prod
$ npm run build.prod
$ npm run serve.prod
# Open your browser at http://localhost:5000/

```
